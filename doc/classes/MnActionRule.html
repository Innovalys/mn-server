\MnActionRule

Manga Network action rule. This action rule is used inside the action router which will select the first rule to match, and performs its action with the current context

The regex that should match the URI. Capturing groups used here could be retrieved by providing capture group entry

The HTTP method that can be used for this rule (get|post|put|delete)
